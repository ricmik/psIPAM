# psIPAM

### Installation 


### A Powershell module for working with phpIPAM.

Start by creating a session to the IPAM server 

```Powershell
New-IPAMSession -ipamserver "ipam.url" -appid "app_id" -username "username" -password "password"
```
#### Available commands
**Get-IPAMIPAddress**

**Get-IPAMSection**

**Get-IPAMSubnet**

**New-IPAMFreeAddress**

**New-IPAMSection**

**New-IPAMSession**

**New-IPAMSubnet**

**Remove-IPAMIPAddress**

**Remove-IPAMSection**
